## Apply a Theme ‚öôÔ∏è

1. Open the command input by pressing `Ctrl(‚åò) + Shift + P` or the `F1` key.

2. Type ¬´*color theme*¬ª in the input field and select `Preferences: Color Theme` from the list.

3. Search for `XulbuX Neon`, `XulbuX PRO` or `XulbuX PRO (Legacy)` and select one to apply it.

## Code used for Theme Previews

#### Python

```python
import random

for IDX in range(10, 0, -1):
    x = random.randint(1, 10) + IDX
    print(f"NUMBER: {x}")

```

#### Java

```java
import java.util.Random;

public class Example {
  public static void main(String[] args) {
    Random random = new Random();
    for (int IDX = 10; IDX > 0; --IDX) {
      int x = random.nextInt(10) + 1 + IDX;
      System.out.println("NUMBER: " + x);
    }
  }
}

```

#### C++

```cpp
#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
  std::srand(std::time(0));
  for (int IDX = 10; IDX > 0; --IDX) {
    int x = std::rand() % 10 + 1 + IDX;
    std::cout << "NUMBER: " << x << std::endl;
  }
  return 0;
}

```

#### Vue.js

```vue
<template>
  <div v-for="i in 10" :key="i">
    NUMBER: {{ randNum() }}
  </div>
</template>

<script setup lang="ts">
const randNum = () => Math.floor(Math.random() * 100);
</script>

```
